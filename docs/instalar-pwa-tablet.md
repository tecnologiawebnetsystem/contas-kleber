# Como Instalar o ContasKleber no Tablet Android

## Passo a Passo para Instalação

### Opção 1: Banner Automático (Recomendado)
1. Abra o app no navegador Chrome do seu tablet
2. Um banner verde aparecerá na parte inferior da tela
3. Clique em "Instalar Agora"
4. Confirme a instalação
5. Pronto! O ícone aparecerá na tela inicial

### Opção 2: Menu do Chrome
1. Abra o app no Chrome
2. Toque no menu (⋮) no canto superior direito
3. Selecione "Adicionar à tela inicial" ou "Instalar app"
4. Confirme o nome e toque em "Adicionar"
5. O app será instalado na tela inicial

## Recursos do App Instalado

### Funciona Offline
- Todas as funcionalidades básicas funcionam sem internet
- Os dados são sincronizados quando a conexão voltar

### Experiência Nativa
- Abre em tela cheia (sem barra do navegador)
- Ícone próprio na tela inicial
- Aparece na lista de apps do Android
- Notificações push (se ativadas)

### Atualizações Automáticas
- O app se atualiza automaticamente
- Sempre terá a versão mais recente
- Sem necessidade de atualizar manualmente

## Testando o App

### No Preview da v0
1. Abra o preview no navegador do tablet
2. O banner de instalação já deve aparecer
3. Teste todas as funcionalidades normalmente

### Após Deploy na Vercel
1. Acesse a URL do app: `https://seu-app.vercel.app`
2. Instale seguindo os passos acima
3. Use como um app nativo!

## Resolução de Problemas

### Banner não aparece?
- Certifique-se de estar usando Chrome ou Edge
- Verifique se já instalou antes (não aparece novamente)
- Limpe o cache e recarregue a página

### App não instala?
- Verifique se tem espaço no dispositivo
- Confirme que está usando HTTPS
- Tente pelo menu do navegador (Opção 2)

### Funcionalidades não funcionam offline?
- Aguarde alguns minutos após a primeira visita
- O service worker precisa fazer o cache inicial
- Abra o app online uma vez antes de usar offline

## Vantagens vs App Nativo

- ✅ Sem necessidade de publicar na Play Store
- ✅ Atualizações instantâneas
- ✅ Funciona em qualquer dispositivo Android
- ✅ Menor tamanho (não ocupa tanto espaço)
- ✅ Desenvolvimento mais rápido e simples
